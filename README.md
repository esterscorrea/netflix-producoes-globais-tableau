# Análise de Produções Netflix por País com Tableau

Este repositório apresenta uma análise visual da distribuição geográfica das produções de conteúdo da Netflix em diversos países. Utilizando o Tableau, foi criado um mapa interativo para explorar e quantificar o volume de filmes e séries associado a cada localidade.

## Objetivo
O objetivo principal desta análise é visualizar quais países contribuem com o maior número de produções para o catálogo da Netflix, oferecendo insights sobre as estratégias de conteúdo e a presença regional da plataforma.

## Ferramentas Utilizadas
- **Tableau Public:** Para a criação do mapa interativo e visualização dos dados.
- **Fonte de Dados:** Um dataset contendo informações sobre os títulos da Netflix (geralmente inclui colunas como `country`, `title`, `release_year`, etc.).

## Visualização
A visualização principal é um mapa coroplético (choropleth map) onde cada país é colorido de acordo com o número total de produções atribuídas a ele. Passar o mouse sobre um país (tooltip) revela o número exato de produções, facilitando a identificação dos principais polos produtores de conteúdo.
